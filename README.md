# frontend-project-lvl2

#### File's Data compare tool. Shows the differences between the objects.

## Setup

```sh
make install
```

## Run tests

```sh
make test
```

## Run

- ### Create a symlink in the global folder

```sh
npm link
```

- ### Use as utility

```sh
gendiff -h
gendiff filepath1 filepath2
```
